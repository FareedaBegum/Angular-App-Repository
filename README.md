# Angular-App-Repository

How to place our code in github:
#steps:

cd app location,
git init,
git add -A,
git commit -m "comment",
git remote add origin https://github.com/FareedaBegum/Angular-App-Repository.git,
git pull -u origin master,
git push -u origin master,
done..
